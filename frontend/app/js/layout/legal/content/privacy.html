Cookies and Web Beacons

A "Cookie" is a string of information that a web server places on a user's device, and that the user's browser provides to the web server each time the user submits a query to the web server. Lockheal uses Cookies and similar and related technologies to track your navigation using the Lockheal Platform, both to help Lockheal improve Users' experiences and to help Lockheal understand how the Lockheal Platform is being used.

A “Web Beacon” is an object that is embedded in a web page that is usually invisible to the user and allows website operators to check whether a user has viewed a particular web page. Lockheal may use Web Beacons or similar technologies in the Lockheal Platform to count Users who have visited particular pages, to compile aggregated statistics about Lockheal Platform usage, and to deliver co-branded services. Web Beacons collect information including a cookie number, time and date of a page view, and a description of the page on which the Web Beacon resides. You may not decline Web Beacons, however, they can be rendered ineffective by declining all cookies or modifying your browser setting to notify you each time a cookie is tendered and permit you to accept or decline cookies on an individual basis. Third parties are not permitted to use Web Beacons on the Lockheal Platform.
